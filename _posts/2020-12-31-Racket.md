CS 135 is a first year computer science class at the University of Waterloo. It is taught using Racket (subset of Scheme) and focuses on recursion.

Topics covered: 

1. Simple Forms of Data
< br >
 2. Structure definitions, constructors, selectors, data definitions, type predicates, functions for mixed data
 < br >
 3. Definition of lists, designing functions to process and produce lists, recursive definition of natural numbers, recursive data definitions.
 < br >
 4. Multi-clausal recursive data definitions, trees, mutually referential data definitions, design methods for complex data, iterative refinement, multiplexing.
 < br >
 5. Encapsulation, block structure, binding, functional abstraction, filters, polymorphism, generic functions, parametric data definitions, functions as data,  functions that produce functions, anonymous functions, abstract list and tree functions (definition and use).
 < br >
 6. Generative versus structural recursion, backtracking algorithms, accumulators and accumulative recursion.
 < br >
 7. History of Computer Science - <a href="https://cs.uwaterloo.ca//current/courses/course_descriptions/cDescr/CS135">Course Website</a>
