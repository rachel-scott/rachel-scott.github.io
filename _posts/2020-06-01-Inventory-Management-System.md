After completing ICS3U, I was fascinated with programming and took ICS4U aka Grade 12 Computer Science.  

One of my assignments was to create an Inventory Management System using Java. 

The program allows user to monitor and update inventory, cost, and price of products.
The program reads in user input to determine which action to perform (add a new product, increase/decrease stock, obtain price/stock of a product, etc). Next the program scans a connected file where all of the inventory data is stored (in arrays). The file is iterated through using loops until the required data has been located or the end of the file is reached. The program updates the file as needed and any requested information is then outputted to the user.
